# Changelog

This project keeps a public changelog so contributors and adopters can see how the shared toolkit evolves over time.

The format is inspired by Keep a Changelog.

## Unreleased

### Changed

- public validation now stays limited to non-sensitive repository, branding, and boundary checks
- comprehensive browser and device evaluation assets are explicitly kept out of the public repository

## 0.1.0 — 2026-03-12

### Added

- browser compatibility matrix for diagnostic modules
- measurement limits guide for refresh-rate, polling, motion, and ghosting interpretation
- permission-handling checklist for media diagnostics
- accessibility audit with shared focus-visible, skip-link, and navigation semantics improvements
- multi-controller gamepad diagnostics with explicit browser-slot selection
- reusable gamepad haptics and vibration capability diagnostics
- locale-aware keyboard legends for common layouts
- mouse drag and hold diagnostics primitives
- monitor motion and ghosting fullscreen pattern utility
- reusable browser API fixtures for diagnostics tests
- GitHub Actions CI, Dependabot, release metadata, and repo governance improvements
